# 🎓 Student Management System

A console-based Student Management System built in Java, demonstrating core Computer Science fundamentals including Object-Oriented Programming, file I/O, and data structures.

## 📋 Features

- ➕ Add new students
- 🔍 Search students by ID or name
- ✏️ Update student information
- ❌ Delete students
- 💾 Persistent data storage using Java serialization
- ✅ Input validation

## 🛠️ Tech Stack

- Java (OOP)
- File I/O (Java Serialization)
- Console/Terminal UI

## 📁 Project Structure

src/
└── com/sms/app/
    ├── Student.java        # Data model
    ├── StudentManager.java # CRUD logic
    ├── FileHandler.java    # File persistence
    └── Main.java           # Entry point & UI

## 🚀 How to Run

1. Clone the repository
2. Open in VS Code
3. Open `Main.java`
4. Click the ▶ Run button
